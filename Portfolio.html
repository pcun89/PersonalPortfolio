Wireless Sensor Network (WNS) - The existing codebase, which can be described as legacy code, was originally written in Swift and later adapted to Java for a research project known as WARP. It aims to enable synchronized program execution on nodes within wireless sensor networks. These programs are crucial for ensuring the reliable and timely transmission of messages from source devices to a central base station, with scheduling options including Priority, Rate Monotonic, or Deadline Monotonic algorithms. While not extensive, the codebase is larger than typical class projects and reflects the practical complexity often encountered in industry settings, where starting from scratch is rare and building on existing legacy foundations is more common.
